# Laundry Mart - Task 2

A simple laundry service website with booking form.

## Features
- Clean and simple design
- Services list
- Price table
- Booking form (Name, Email, Phone)
- Thank you message
- Mobile responsive

## How to Run
1. Download both files
2. Put them in same folder
3. Open index.html in browser
