# Laundry Mart - My Second Task

## About This Project
This is my second task. I made a website for a laundry service called "Laundry Wallah".

## What I Learned
- How to make tables with borders (border="1" in HTML and border in CSS)
- That every input needs a "name" attribute (my mentor told me).
- How to make buttons change color when you hover
- That CSS takes a LOT of trial and error

## What Was Hard
The table borders didn't show up at first. I searched on Google "html table border not working" and found out I needed border="1" in the table tag AND border in CSS. It took me 30 minutes to figure this out!

The button hover also didn't work because I forgot to style the button first. I only had .submit-btn:hover without .submit-btn.

## Colors I Choose
I used orange (#ff8c42) because it's looks nice. I think it looks good for a laundry service - warm and friendly.
